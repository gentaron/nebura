事件・時代一覧

E16星系の歴史を形作った重要な出来事と時代区分の詳細な記録です。

歴史的時代区分

E16星系の起源

西暦3500年 / E1年 植民開始期

地球からの大移民によるE16星系植民開始。ティムール・シャーのペルセポネ計画と、星々の交響曲・Eros-7の初期開発が始まった時代。

技術啓蒙時代

西暦3580-3590年 / E80-E90年 技術革新期

テミルタロンの次元極地平開発とテクノ宗教運動の台頭。バイオエンジニアリングの発展とプライマリー・フィールド構築により、E16星系の技術基盤が確立された。

第三次繁栄

西暦3597-3601年 / E97-E101年 人口急増期

人口1億2000万人を突破し、コーポラタムパブリカとA籍制度が確立。同時期、Eros-7では搾取生物の危機とマトリカル社会の基盤が形成された。

第四次繁栄

西暦3653-3701年 / E153-E201年 経済黄金期

マーストリヒト革命後の経済黄金期。人口3億人突破、年間2京nトークンの経済規模を達成し、M104銀河最大の経済ハブとして確立された。

重要な政治・社会事件

地球からの大移民

西暦3500-3501年 / E1年 移住事件

人口過剰と資源枯渇に直面した地球から、曲率航法と量子テレポーテーション技術により、多様な民族集団がE16星系への移住を開始した歴史的事件。

マーストリヒト革命

西暦3650年 / E150年 政治革命

エル・フォルハウス指導による旧バース帝国の貴族制打倒。完全自由経済システムの確立とコーポラタムパブリカ成立をもたらした歴史的転換点。

ニューヘルシンキ宣言

西暦3651年 / E151年 政治宣言

アリア・ソルが提唱した E16星系統合構想。惑星連邦または世界政府の設立を目指したが、企業国家連合の反対により部分的実現に留まった。

多元文化協定

西暦3501年 / E1年 文化協定

星々の交響曲でティムール・シャーの指導により結ばれた、フェンドラ、アーク、ポロンポロ3民族の文化融合協定。E16星系の文化的多様性の基盤となった。

危機・災害事件

搾取生物の危機

西暦3600年 / E101年 生物災害

Eros-7でバイオリアクターの制御失敗により搾取生物が異常増殖。シルヴィア・クロウのエスパー能力により解決され、マトリカル社会の基盤となった危機的事件。

コーラの疫病

西暦3708年 / E208年 疫病災害

アンドロメダ系移民の遺伝子に特異的に作用するウイルス感染。人口の15%が死亡し、高階層・低階層間の医療格差が社会分断を深刻化させた。

文化・宗教・社会運動

テクノ宗教運動

西暦3580年代 / E80年代 宗教運動

次元極地平の成功により触発された科学と神秘主義の融合運動。テミルタロンを精神的支柱とし、技術発展を宗教的体験として昇華させた。

戦士決定戦

第三次繁栄期以降 競技・社会制度

ネオンコロシアムで開催される実力主義競技大会。低階層者がA籍昇格を目指し、ナノアーマーとプラズマブレードで戦う個人主体性の象徴的イベント。

フェスティバル・オブ・フレア

搾取技術発展期以降 文化祭典

Eros-7で開催される搾取技術とマトリカル社会を祝う最大の祭典。搾取触手を使った舞踏とバイオ発光による光のショーで、女性の優位性を強化する。

星間芸術祭

植民初期以降継続 芸術イベント

星々の交響曲で開催される文化的統合イベント。プライマリー・フィールド内で仮想アートや音楽が創作され、移民の文化的アイデンティティを強化している。

抵抗運動・社会変革

シャドウ・リベリオン

コーラの疫病以降 反体制運動

星々の交響曲の低階層者による反体制組織。A籍制度による格差と医療不平等に抗議し、戦士決定戦を利用して政治的メッセージを発信している。

シャドウ・ユニオン

搾取生物危機以降 男性解放運動

Eros-7の男性労働者による秘密結社。搾取技術の廃止と男性解放を目標とし、アビス・チェンバーで秘密集会を開催している地下抵抗組織。
