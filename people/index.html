人物一覧

E16星系の歴史を形作った重要人物たちの詳細な記録です。

ティムール・シャー

西暦3500年代 / E1年代 移民団リーダー・理論物理学者

10次元ホラズム理論の提唱者であり、E16星系への最初の移民団を率いた指導者。ペルセポネシステムを設計し、E16星系植民の基礎を築いた。

テミルタロン

西暦3580年代 / E80年代 物理学者・テクノ宗教指導者

サイケデリック・コスモロジーを提唱し、次元極地平技術を開発。テクノ宗教運動の精神的支柱として、科学と神秘主義の融合を推進した。

エル・フォルハウス

西暦3650年代 / E150年代 革命家・経済学者

「新時代のルーキー」として知られるマーストリヒト革命の指導者。貴族制を打倒し、自由経済システムとコーポラタムパブリカを確立した。

アリア・ソル

西暦3650年代 / E150年代 政治家・統合主義者

エル・フォルハウスの盟友で、ニューヘルシンキ宣言を提唱。E16星系全体の統合を目指し、多元文化協定の継承者として活動した。

リリス・ヴェイン

西暦3520年代 / E20年代 Eros-7初代女性リーダー

搾取生物を「宇宙の贈り物」として積極的に活用し、Eros-7における搾取技術の基礎を築いた初代女性リーダー。バイオリアクター技術の開発者。

エラナ・セレス

西暦3580年代 / E80年代 バイオエンジニア・第二世代女性リーダー

リリス・ヴェインの後継者として、搾取ヒルと媚薬ガスの実用化を推進。女性評議会を設立し、プロト・レジストリ（精子レジストリの原型）を導入した。

シルヴィア・クロウ

西暦3600年頃 / E101年頃 エスパー能力者・生物学者

搾取生物の危機を解決したエスパー能力者。テレパシーとエネルギー操作能力で搾取生物を制御し、マトリカル社会の基盤を確立した。

マリア・フォルハウス

西暦3650-3701年 / E150-E201年 遺伝子工学者・搾取技術革新者

エル・フォルハウスの子孫で、搾取技術を洗練化。触手型搾取生物や搾取バクテリアを開発し、Eros-7を星間エネルギー輸出基地に発展させた。
